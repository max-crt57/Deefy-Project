# Deefy

Deefy est une application web en PHP permettant de créer, gérer et écouter des playlists musicales et des podcasts.
Projet réalisé dans le cadre du BUT Informatique - IUT Nancy-Charlemagne (2025).

## Fonctionnalités principales

*  Connexion / Inscription / Déconnexion des utilisateurs
*  Création et gestion de playlists personnelles
*  Ajout et lecture de morceaux ou podcasts (.mp3)
*  Rôle administrateur (accès à toutes les playlists)
*  Stockage MySQL avec PDO et requêtes préparées
*  Sécurisation des entrées et gestion des erreurs
